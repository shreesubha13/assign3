CSS

CSS stands for Cascading Style Sheets. CSS is the language for describing the presentation of web pages, including colors, layout, and fonts. CSS describes how HTML elements are displayed on screen, paper, or in other media. CSS saves a lot of work. This style sheet language also allows you to add effects or animations to your website.

Three types of CSS styles:

  • Inline style
  • Internal Style
  • External Style
